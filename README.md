# CM Design Pubzi Redesign

Portafolio Next.js rediseñado con una estética esports oscura y tecnológica, conservando la lógica editable, las rutas públicas, la galería, la tienda y el panel administrativo del proyecto original.

Consulta [`REDISENO_UX_UI.md`](./REDISENO_UX_UI.md) para ver el alcance completo, instrucciones de instalación y rutas principales.

## Inicio rápido

```bash
npm install
npm run dev
```
