# Portafolio CM Design + Payload CMS

Este portafolio ahora incluye un **panel de administración completo** con Payload CMS.

## Cómo usar el Admin

1. `npm install`
2. `npm run dev`
3. Ve a `/admin` para acceder al panel
4. Crea tu primer usuario administrador

Todo el contenido (proyectos, sobre mí, arsenal) se puede editar desde el panel sin tocar código.
